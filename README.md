# ScreenSense: Kids Screen Time Analysis

This project analyzes screen time patterns among children using the Indian Kids Screen Time dataset.

## Project Structure

data/
- raw dataset
- processed dataset

notebooks/
- data understanding
- preprocessing
- analysis

outputs/figures/
- saved charts

## Libraries Used

pandas  
numpy  
matplotlib  
seaborn  

## Key Insights

- Teenagers (14–17) show the highest screen usage.
- Smartphones are the primary device driving screen exposure.
- Most children spend between 3–6 hours daily on screens.
- Screen time patterns are similar across genders.
- Certain age-device combinations show higher risk of excessive screen time.
